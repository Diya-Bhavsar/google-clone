# 🔍 Google Homepage Clone

A responsive **Google Homepage Clone** built using **HTML, CSS, and JavaScript**. This project was created for educational purposes to enhance my frontend development skills by recreating the clean and minimal design of Google's homepage.

## 🚀 Live Demo

🔗 https://diya-bhavsar.github.io/google-clone/

## 📂 GitHub Repository

🔗 https://github.com/Diya-Bhavsar/google-clone

## ✨ Features

- Clean and responsive Google homepage UI
- Search bar interface
- Navigation links
- Google Apps icon
- Profile icon
- Footer similar to Google's homepage
- Responsive layout
- Hover effects

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 🎯 Purpose of the Project

This project was built to:

- Practice building modern user interfaces
- Improve HTML and CSS skills
- Learn responsive web design
- Strengthen JavaScript fundamentals
- Recreate a real-world website for educational purposes
